# Node Js Backend with MySQL, Sequelize, and Docker

This project uses:

- Node Js
- Express
- MySQL
- Sequelize

## Installation

```bash
git clone https://github.com/andymyp/API-Express-MySQL-Docker.git
```

```bash
cd API-Express-MySQL-Docker
```

```bash
npm install
```

```bash
docker-compose up -d
```
